### Configuration

go to `path/to/config.php` to set all API configuration.

## Installation

### With Composer

```
$ composer require ahmedsaoud31/payfort-php
```
or

```json
{
    "require": {
        "ahmedsaoud31/payfort-php": "~1.0"
    }
}
```
use

```php
<?php
require 'vendor/autoload.php';

use Payfort\Payfort;

$payfort = new Payfort;
```

### Without Composer

```php
<?php
require 'path/to/Payfort.php';

use Payfort\Payfort;

$payfort = new Payfort;
```
